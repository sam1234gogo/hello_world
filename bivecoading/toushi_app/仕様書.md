# 株式投資アシスタント 仕様書

## 1. アプリ概要

### 概要
株式投資アシスタントは、日本株（TOPIX Core30・Large70銘柄）の投資判断をサポートするFlask Webアプリケーションです。短期・中期・長期の3種類の投資ロジックに基づいてスクリーニングを実行し、投資判断の参考情報を提供します。

### 主な機能
- **自動スクリーニング**: 全70銘柄をリアルタイムでスコアリング・分類
- **ポートフォリオ管理**: 購入・ナンピン・売却の記録と損益計算
- **アラート機能**: ナンピン水準・売り時・期限切れを自動通知
- **バックテスト**: 過去データでの戦略検証
- **設定管理**: 日経平均PBR・目標利回り等の手動設定

### 2.1 短期投資ロジック（最重要・最優先表示）

#### 買いトリガー
- EPS年間予想が前回比 **+10%以上の上方修正**

#### スコアリング（5項目、最高+4点×5=+20点）

| 指標 | 条件 | 点数 |
|------|------|------|
| PBR | ≤0.5倍 | +4 |
| PBR | ≤0.8倍 | +2 |
| PBR | ≥1.0倍 | -3 |
| PER(予想) | ≤10倍 | +4 |
| PER(予想) | ≤15倍 | +2 |
| PER(予想) | ≥20倍 | -3 |
| ROE(予想) | ≥16% | +4 |
| ROE(予想) | ≥8% | +2 |
| ROE(予想) | ≤2% | -3 |
| 配当利回り | ≥4% | +4 |
| 配当利回り | ≥3% | +2 |
| 配当利回り | ≤1.5% | -3 |
| 52週安値比 | ≤1.2倍 | +2 |
| 52週安値比 | ≥1.5倍 | -3 |

#### 区分判定
```
合計+2点以上 かつ 経常利益増益
  → PBR≤0.75 または 配当利回り≥2.4%
      → 安心割安株 ★最優先表示
  → それ以外
      → 通常割安株

合計-10点以下
  → 成長株（ROE極端に高い）

中間
  → スキップ
```

#### ナンピン水準とナンピン量
| 水準 | 安心割安株 | 通常割安株/成長株 |
|------|-----------|-----------------|
| -10% | ×1 | ×2 |
| -15% | ×1 | ×3 |
| -20% | ×1 | ×4 |

#### 売り時
| 条件 | 売り価格 |
|------|---------|
| 通常 | 買い値 × 1.09（+9%） |
| ナンピン後 | 買い平均 × 1.10（+10%） |
| 3ヶ月経過 | 強制売却（安心割安株は中期/長期移行） |
| ロスカット | 買い値 × 0.75（-25%） |

### 2.2 中期投資ロジック

#### 買いトリガー
個別銘柄PBR ≤ 日経平均PBR × 係数

| 企業種別 | 黒字時係数 | 赤字時係数 |
|---------|-----------|----------|
| 国際優良企業 | × 0.6 | × 0.3 |
| 財務優良企業 | × 0.5 | × 0.25 |

#### サポート/レジスタンス計算
過去5年の週足データから価格クラスタリングで算出：
- **Support1**: 過去で最も頻繁に反発した安値圏
- **Support2**: Support1を下抜けた際の大底価格帯
- **Resistance1**: 過去の高値圏

#### 状態管理
| 状態 | トリガー | アクション |
|------|---------|---------|
| 監視中 | 現在価格 ≤ Support1 | 第1購入通知 |
| 保有中(1次) | 現在価格 ≥ Resistance1 | 利確通知 |
| 保有中(1次) | 現在価格 ≤ Support2 | ナンピン通知 |
| 保有中(ナンピン済) | 現在価格 ≥ Resistance1×1.1 | 利確通知 |

---

### 2.3 長期投資ロジック

#### 対象条件
| 条件 | 値 |
|------|-----|
| 過去20年赤字回数 | 2回以下（手動確認） |
| 安定株主率 | ≥40%（手動設定） |
| 配当 | 1株30円以上 |
| 減配履歴 | 過去15-20年なし（手動確認） |
| 日平均出来高 | ≥50,000株 |

#### 買い目標株価計算
```
買い目標株価 = 予想年間配当 ÷ 目標配当利回り(3〜5%)
```

#### ナンピン
- 初回購入から -10%、-20% で同数量追加

#### 売り時
```
売り目標 = 最安値 × (1 + 目標利回り × 10)
```
※目標利回り4%の場合: 最安値 × 1.4（+40%）

## 3. 画面仕様

### 3.1 ダッシュボード（/）
メインスクリーニング結果画面。

**表示セクション（上から順）:**
1. ★★★ 短期 安心割安株（金色カード、Glow効果）
2. 短期 通常割安株・成長株（青カード）
3. 中期投資候補（緑カード）
4. 長期投資候補（紫カード）

各セクションにBootstrapレスポンシブテーブル。

### 3.2 ポートフォリオ（/portfolio）
保有株管理画面。

**表示セクション（上から順）:**
1. アラート（ナンピン推奨・売り時を最上部に強調表示）
2. 保有株一覧テーブル
3. 売却済み銘柄（直近10件）

**機能:**
- 銘柄追加、ナンピン記録、売却記録（モーダルダイアログ）
- リアルタイム損益計算
- 3ヶ月期限の残日数表示

### 3.3 シミュレーション（/simulation）
バックテスト画面。

**機能:**
- 期間・銘柄・戦略・初期資金の設定
- Plotlyによる損益曲線グラフ
- トレード一覧テーブル
- 勝率・最大DD等の統計表示

### 3.4 設定（/settings）
アプリ設定管理画面。

**設定項目:**
- 日経平均PBR（手動入力）
- 目標配当利回り範囲（最小・最大）
- スクリーニング対象（Core30のみ/Core30+Large70）
- データ更新頻度
- 銘柄ごとの手動設定（海外売上高比率・安定株主率）

## 6. 設定方法

### 6.1 インストール手順
```bash
# 依存パッケージのインストール
cd /Users/sam/bivecoading/toushi_app
pip install -r requirements.txt

# アプリの起動
python app.py
```

### 6.2 アクセス
ブラウザで `http://127.0.0.1:5000` を開く

### 6.3 初回設定
1. 設定画面（/settings）で日経平均PBRを最新値に更新
2. スクリーニング対象銘柄（Core30/Core30+Large70）を選択
3. 目標配当利回り範囲を設定（デフォルト: 3〜5%）
4. 各銘柄の海外売上高比率・安定株主率を手動入力

### 6.4 データ更新
- 自動: 1時間ごとにキャッシュが更新される
- 手動: ダッシュボードの「データ更新」ボタンをクリック

## 8. 制限事項・免責事項

### データの制限
- yfinanceは非公式APIのため、データに欠損・誤りが生じる場合があります
- リアルタイムデータではなく、通常15〜20分の遅延があります
- EPS改訂率の追跡は、アプリ使用開始後にデータが蓄積されるまで機能しません
- 中期投資のサポート/レジスタンス計算は簡易的なアルゴリズムです
